# pidgin-numix-icon-theme
Unofficial Numix based Icon Theme for Pidgin

## Installing
- Copy the numix and numix-square folder into ~/.purple/themes/
- Restart Pidgin
- Go to Tools > Preferences > Themes
- Select the new status icon theme
